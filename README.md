How to run the app:
1.Launch Android Studio from your desktop or start menu.
2. Click "Open" on the welcome screen. If already in Android Studio, go to File then Open from the top menu.
3. Browse to your Piggy Pilot project folder on your computer.
4. Select the project folder
5. Wait for Gradle to sync
5. After it sync you can
6. Run the app, click the green triangle
